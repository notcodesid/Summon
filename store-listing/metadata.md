# Summon store listing draft

## Name

Summon

## Short description

Pull a collectible, then check the random roll that picked it.

## Full description

Summon is a small collectible pull game built on Solana Devnet.

Tap once to discover an item. Each result comes from verifiable randomness, and the app shows the roll, rarity, transaction, and settlement proof behind it. Your collection is read from your on-chain player account rather than filled with demo items.

Summon uses MagicBlock Ephemeral Rollups to keep the pull flow quick while the result settles back to Solana. Sign in with Google and an embedded Solana wallet is created for you.

This release is an early Devnet demo. Collectibles have no promised monetary value and cannot currently be bought, sold, or traded in the app.
